# 🌾 KrishiGuard AI - Crop Disease Detection

Check out Here → 
---

KrishiGuard AI is an AI-powered crop disease detection system designed to help farmers quickly identify plant diseases through leaf image analysis. Using **Deep Learning & Computer Vision**, it provides accurate and real-time results, enabling early intervention to prevent crop loss.

---

## 🚀 Features
✅ AI-powered plant disease detection  
✅ Mobile-friendly interface for easy access 📱  
✅ Real-time disease diagnosis with **98.7% accuracy**  
✅ Supports **38+ plant varieties**  
✅ Instant results with deep learning-based predictions  
✅ Severity rating & treatment recommendations per disease  
✅ User-friendly web interface powered by **Streamlit**  

---

## 📷 How It Works?
1️⃣ **Capture** - Take a clear photo of a plant leaf 📸  
2️⃣ **Upload** - Submit the image on the Disease Finder page  
3️⃣ **Analyze** - The AI model processes the image using deep learning  
4️⃣ **Results** - Get instant diagnosis, confidence score & treatment tips  

---

## 🛠️ Tech Stack
- **Frontend:** Streamlit (Python-based UI)  
- **Backend:** TensorFlow, Keras, NumPy  
- **Machine Learning Model:** Convolutional Neural Networks (CNN) — 16-layer architecture  
- **Deployment:** Streamlit Cloud / Local Server  

---

## 📂 Dataset Information
- **Source:** [Plant Diseases Dataset](https://www.kaggle.com/datasets/vipoooool/new-plant-diseases-dataset) 🌱  
- **Images:** 87,000+ labeled plant leaf images  
- **Classes:** 38 plant disease categories  
- **Train Split:** 70,295 images (80%) | **Val Split:** 17,572 images (20%)  
- **Augmentation:** Rotation, flipping, zooming for better model generalization

## 🚀 Installation Guide

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/kunalvaishkiyar/KrishiGuard-AI.git
cd KrishiGuard-AI
```

### 2️⃣ Install Dependencies
```bash
pip install -r requirements.txt
```

### 3️⃣ Run the App
```bash
streamlit run web.py
```
